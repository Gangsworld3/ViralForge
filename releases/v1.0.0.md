# ViralForge AI v1.0.0

## Release Summary

ViralForge AI 1.0.0 is the first stable release of the local-first short-form content automation pipeline. This release consolidates the project into a publishable package with hardened runtime behavior, SQLite-backed operational state, packaged entrypoints, and a cleaner release surface.

## Highlights

- SQLite-backed runtime state instead of fragmented JSON state for core operations
- Stable package entrypoint through `python -m viralforge`
- State database maintenance commands:
  - `state-db --summary`
  - `state-db --integrity-check`
  - `state-db --backup`
  - `state-db --export-json`
  - `state-db --restore`
  - `state-db --vacuum`
- Posting, analytics, memory, and video-planning hardening
- Streamlined README, changelog, and release metadata

## Validation

- `python -m compileall .`
- `python -m unittest discover -s tests`
- `python -m viralforge state-db --summary --integrity-check`
- `python -m viralforge demo --topic "AI productivity hacks"` in smoke/manual mode
